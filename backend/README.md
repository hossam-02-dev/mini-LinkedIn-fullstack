# 🚀 miniLinkedin - Plateforme Académique Professionnelle

**miniLinkedin** est une application web full-stack conçue pour digitaliser les interactions au sein de l'écosystème académique. Ce projet permet de connecter étudiants et professeurs tout en offrant des outils de communication modernes.

## 🛠️ Stack Technique

### Frontend
- **Framework** : React + Vite
- **Styling** : Tailwind CSS 4 (Design moderne et mature)
- **Communication** : Axios (gestion du CORS pour la liaison Backend)

### Backend
- **Framework** : Spring Boot
- **Sécurité** : Spring Security + JWT (Authentification Stateless)
- **Temps réel** : Spring WebSockets (STOMP/SockJS) pour la messagerie
- **Base de données** : PostgreSQL

---

## 🔐 Fonctionnalités Clés

- **Authentification Sécurisée** : Système basé sur les rôles (Etudiant, Professeur) avec protection JWT.
- **Messagerie Instantanée** : Chat en temps réel via WebSockets.
- **Gestion Académique** : CRUD complet pour les formations et les profils.
- **Documentation API** : Intégration de Swagger UI pour tester les endpoints.
